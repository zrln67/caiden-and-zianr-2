# Project Title: Smart Budget Planner

## Project Description
This is a Python program the helps people track their expense so they dont overspend, saving a lot of money.

## Features
- Sign in or make account
- Data will be saved
- Enter transaction
- Calculates their updated budget
- Enter how much your item cost and enter item names
- Display transaction history
- Enter bills and due payments

## How to Run the program
1. Download the the version you want for Smart Budget Planner and extract it
2. Open any ide
3. Open the in the current ide you are in folder
4. Follow the instrunctions inside "how to run"
